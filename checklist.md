# imgrab 验收检查清单

## 功能验收

### 镜像拉取功能

- [ ] 支持从 Docker Hub 拉取镜像
- [ ] 支持从私有仓库拉取镜像
- [ ] 默认与当前运行环境架构一致
- [ ] 支持手动指定架构（--arch 参数）
- [ ] 支持使用 `@` 指定镜像版本（如 `nginx@latest`）
- [ ] 支持保存镜像 tar 文件到指定目录（--output 参数）
- [ ] 支持拉取后自动导入 Docker（--import 参数）
- [ ] 默认只下载不导入
- [ ] 拉取时有进度条展示
- [ ] 镜像分层并行拉取（可选优化）

### 认证功能

- [ ] 支持登录 Docker Hub
- [ ] 支持登录私有仓库
- [ ] 凭证信息安全存储

### 搜索功能（TUI）

- [ ] 支持搜索镜像
- [ ] TUI 界面展示搜索结果
- [ ] 镜像按版本列出
- [ ] 选中版本后（Enter 键）弹窗确认下载
- [ ] 弹窗中可选择架构
- [ ] 默认架构为当前运行环境架构

## 技术验收

- [ ] 使用 Go 1.25.7 版本
- [ ] 使用 github.com/google/go-containerregistry 库
- [ ] 代码结构清晰，模块化设计
- [ ] 错误处理完善
- [ ] 无明显性能问题

## 文档验收

- [ ] README.md 包含使用说明
- [ ] 命令帮助信息完整（--help）
